# Currency converter
Una convertidor de moneda hecho con jQuery y [nice-select](https://github.com/hernansartorio/jquery-nice-select). 
### Se puede convertir a:
* Dolar 
* Reales
* Peso argentino
* Peso chileno
* Euros
* Libras
* Bitcoins

Visita: [gkoutian.github.io/currency](http://gkoutian.github.io/currency)
